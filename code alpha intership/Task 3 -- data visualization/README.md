# CodeAlpha Task 3 - Data Visualization

## Objective
The objective of this project is to visualize book data using Python visualization libraries and gain meaningful insights from the dataset.

## Dataset
- File: books.csv
- Total Records: 20
- Columns:
  - Title
  - Price
  - Rating

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Rating Distribution
- Price Distribution
- Price Box Plot
- Average Price by Rating

## Insights
- The dataset contains books with different ratings.
- Most books fall into a particular rating category.
- Book prices vary across the dataset.
- Average price differs slightly among rating categories.
